Release Notes

VisualVM 2.1.10 has been released on September 17th, 2024. This document describes changes introduced by this release.

System Requirements

Supported OS:

  • Microsoft Windows
  • Linux: Intel platform, ARM HFLT, AArch64
  • macOS: Intel platform, Apple Silicon

Required Software:

  • Oracle JDK 8~23
  • OpenJDK 8~23
  • GraalVM 19 ~ GraalVM for JDK 23

Changes

Features and Enhancements:

  • GH-597: Upgrade NetBeans Platform to 22
  • GH-590: Add support for JDK 23
  • GH-439: [Threads tab] Ability to show only running or sleeping or waiting, threads

Fixed Bugs:

  • GH-605: Wrong macOS in JFR Viewer
  • GH-599: [HeapViewer] Slow computation of retained size
  • GH-598: VSCode Extension - Source Roots don't work in VSCode extension
  • GH-596: [HeapViewer] Do not expand all threads
  • GH-595: [HeapViewer] Cannot expand 'null' node
  • GH-524: Problems Displaying OS in JFR Viewer

Known Problems

To view all known problems please see the VisualVM Issues page.

To resolve the most common problems please follow the Troubleshooting Guide.
